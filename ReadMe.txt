Steps On running the RMI server


Make sure the RMI server is running on localhost.
deploy the servlet on application server in this case JBOSS
Initiate an request using client mobile app (http request using the localhost  IPv4 Address. . . . . . . . . . . : 192.168.100.2)
This is the source folder for RMI server and mobile client.RMI provided remote communication between Java programs.The javax.servlet and javax.servlet.http packages provided interfaces and classes for writing servlets. The HttpServlet class provided methods, such as doGet and doPost, for handling HTTP-specific services.The program was then hosted on JBOSS application server. client requests were initiated from the mobile app.
